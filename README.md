## Empezando

Para iniciar, y ejecutar las pruebas unitarias, seguir los siguientes pasos:

## Clonar repositorio

```bash
# Clonar Repositorio
$ git clone https://github.com/icalre/lambda-test.git

# Ir al directorio
$ cd lambda-test

# Instalar depencias
$ npm install
```

## Configurar .env

```bash
# Ir al directorio
$ cd lambda-test

# Copiar .env.example
$ cp .env.example .env

# Reemplazar las cariables del .env por los suyos 

```

## Ejecutar Localmente

```bash
# Ir al directorio:
$ cd lambda-test/
# Compilar y exponer servicios localmente
$ npm run dev
```

## Test Localmente

```bash
# unit tests
$ npm run test
```
